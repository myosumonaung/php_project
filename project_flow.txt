Blog
=====

Features
------------
- User(reader, writer, admin)
- Posts
    - category


Posts
------
- View Posts -> reader[no auth]
- Create Posts -> writer[auth]
- Public Posts -> admin[auth] (public, reject)


User Journey
-------------
Reader -> Can view all Posts
       -> detail view
       -> filter by category

Juu    -> Register
       -> Login
       -> Profile
       -> my own posts (create, public, reject)
       -> create, edit, delete, retrieve

Mg Mg  -> Login
       -> all posts <- writer's created posts
       -> detail view
       -> Reject (reason)


Category [hardcode]
    - Create, retrieve, update, delete

1. DB create
    - category
        - id [PK, AI]
        - name [varchar]

    - users
        - id [PK, AI]
        - name [varchar]
        - email [varchar, UNIQUE]
        - password [varchar] <- hash
        - role (write, admin) [varchar]

    - posts
        - id [PK, AI]
        - title [varchar]
        - content [text]
        - author_id [FK]
        - category_id [FK]
        - status [created, published, rejected] -> [varchar]
        - published_at [timestamp, NULL]
        - created_at [timestamp]
